# V后台管理系统模板

#### 项目介绍
借鉴林鑫朋友的后台管理系统进行再创造，这是林鑫的git地址https://github.com/lin-xin。
感谢林鑫，这套模板以后也会不断维护，尽量做到适应大部分企业后台管理系统，大家的支持是我的动力

#### 软件架构
主要以vue+element为主构建


#### 安装教程

1.
克隆本项目 
git clone https://gitee.com/Q_Augly/VueTemplate.git
2.
安装依赖
npm install
3.
运行项目 
npm run dev
